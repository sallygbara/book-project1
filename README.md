# My Book Archive 📚

אפליקציית React + TypeScript לניהול ארכיון ספרים אישי (CRUD) עם MockAPI.  
כולל: הוספה, עריכה, מחיקה, חיפוש לפי כותרת, וסימון מועדפים.

---

## ✨ פיצ'רים
- ✅ הצגת ספרים (GET)
- ✅ הוספת ספר חדש (POST)
- ✅ עריכת ספר (PUT)
- ✅ מחיקת ספר (DELETE)
- ✅ חיפוש לפי כותרת
- ✅ מועדפים (toggle `isFavorite`)
- ✅ עיצוב עם Tailwind + Flowbite React
- ✅ תמונת עטיפה לפי URL (כולל fallback אם כתובת לא תקינה)

---

## 🧱 טכנולוגיות
- React + TypeScript (Vite)
- TailwindCSS
- Flowbite React
- MockAPI (REST)
